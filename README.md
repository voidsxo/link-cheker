# link-cheker
🧠 بیوگرافی ابزار BURGERR یک ابزار سبک اما هوشمند برای تحلیل لینک و تشخیص تهدیدهای احتمالی در سطح دامنه است. این ابزار با ترکیب طراحی بصری مدرن و یک موتور تحلیل مبتنی بر قواعد (Rule-Based)، قادر است به‌صورت سریع وضعیت امنیتی یک لینک را بررسی کرده و احتمال فیشینگ یا جعلی بودن آن را مشخص کند.
